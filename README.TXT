Projeto: visor-relogio
Autores: David Barnes and Michael Kölling
Tradução: Daniel Conrado

Este projeto é parte do material para o livro

   Objects First with Java - A Practical Introduction using BlueJ
   Sixth edition
   David J. Barnes and Michael Kölling
   Pearson Education, 2016

Ele é discutido no capítulo 3.

Para usar este projeto, crie uma instância da classe VisorRelogio. Depois inspecione
esta instância e mantenha a janela de inspeção aberta. Com essa janela aberta,
chame os métodos do objeto, como tique e setTempo. Observe o campo
"stringExibição" no Inspetor.

O campo stringExibição simula o visor do relógio real. Se este programa estivesse
rodando em um relógio de verdade, ele mudaria o visor físico (o componente
eletrônico) ao invés desta string. O método tique seria disparado uma vez a cada
minuto por algum temporizador físico.

Todo o resto seria praticamente igual.

Nota: Este projeto não implementa tratamento adequado de erros.
